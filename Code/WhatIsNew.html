What's new in 24U Plug-In Template 3.0

The following changes has been introduced with the release of the version 3.0 of 24 Plug-In Template:

  • project renamed to 24U Plug-In Template
  • XCode project adjusted for Mac OS X 10.5 and 10.6 preserving compatibility with 10.4
  • added project for Microsoft Visual Studio 2010
  • FMPlugInSDK 11 added to the project
  • formatted documentation
  • added "Template Version.fp7" example
  • X24FT_Version function works with no parameters
  • X24FT_Version function supports "autoupdate" parameter to better distinguish between plug-in versions
